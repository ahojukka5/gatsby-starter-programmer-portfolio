This post is about testing PrismJS code formatting with various programming languages.

Using code formatting follows markdown specification, i.e. triple backticks and optionally programming language given. Optionally, lineNumbers can be given in the first line after programming language, to turn on line numbering.

```python lineNumbers
print("Hello, world!")
```

The markdown above is rendered as follows:

print("Hello, world!")

Python

import datetime

print("Now is ", datetime.datetime.now())

def hello(name: str, count: int) -> int:
    for i in range(count):
        print(f"Hello, {name}")
        if i > 3:
            break
    return len(name)
